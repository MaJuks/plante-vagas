✅ Pré-requisitos
Antes de começar, certifique-se de ter o Node.js instalado em sua máquina. Você pode baixá-lo em: https://nodejs.org/

🚀 Como rodar o projeto localmente

Siga os passos abaixo:

Clone o repositório para sua máquina:

- git clone https://gitlab.com/plante-vagas/plante-vagas-v2.git

Acesse a pasta principal do projeto:

- cd plate-vagas-v2

Entre na pasta do front-end (client):

- cd client

Instale as dependências do projeto:

- npm install

Inicie o servidor de desenvolvimento:

- npm run dev.
